Executive Summary

Bridging the gap from Content Obesity to Intent-Driven UX.

The Transformation Map

Current State (The Gap)

  • Geographic Navigation

    Content buried under "Region > Country > Office." Hard to find global services.

  • Internal Jargon

    Using terms like "B2G," "SME Catalyst," or "FDI" which alienate new users.

  • Passive PDF Library

    A "Report Graveyard" of 100+ static files. Browsing leads to "Analysis Paralysis."

  • Invisible Tools

    High-value calculators and search bars are hidden behind icons or sub-menus.

Future State (The Goal)

  • Intent Navigation

    Organized by: "I want to sell," "I want to invest," or "I need to comply."

  • User Language

    Action-oriented outcomes like "Winning Public Tenders" or "Scale-up Support."

  • Dynamic Knowledge Hub

    Searchable, filterable insights database with clear "Next Steps" calls-to-action.

  • Action-Oriented UX

    Prominent open search bars, diagnostic quizzes, and interactive trade tools.

The Core Challenge

Current data confirms "Content Obesity." The site has grown organically without pruning, resulting in buried services and high bounce rates. Users looking for specific outcomes (e.g., "NATO Tenders") are lost in geographic silos.

The Strategic Pivot

Shift from "Who We Are" (Geographic/Internal Depts) to "What You Need" (Jobs-to-be-Done). We propose a 3-pillar pillar structure: 1. Grow Global, 2. Invest in Sweden, 3. Market Intelligence.
